## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/NamanOG/Netflix-Clone.git
    ```

2. Open the project folder:
  
    ```bash
    cd Netflix-clone
    ```

3. Open `index.html` in your preferred web browser.
